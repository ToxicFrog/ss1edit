These maps are generated from the System Shock ARCHIVE.DAT file using ss1edit. Use the controls on the left to select which object types to display and switch levels; mouse over the map for information on map tiles and objects. Use the search bar on the right to find objects by name. Mouse over a search result to hilight it on the map and show the corresponding object information; click on one to switch to the map that contains it.

The map itself is 1024x1024, and the controls on either side of it add some width. You will probably want at least at 1680x1050 display to use it comfortably.

These are "unified" maps. They are large ("all levels" is 4MB) and have higher memory requirements than individual maps, but let you quickly switch between levels.
